# WiFi Finder 서비스 시장 조사 (Survey)

> **작성일**: 2026-05-12  
> **목적**: 사용자들이 WiFi 가능 지점을 커뮤니티 기반으로 공유하는 서비스 구축을 위한 시장 조사 및 유사 서비스 분석

---

## 1. 시장 개요

### 1.1 문제 정의
- 모바일 데이터는 유한하며, 특히 원격 근무·여행·외출 시 WiFi 접속이 필수적
- 공공 WiFi 위치 정보가 분산되어 있고, 실시간 상태(동작 여부, 속도, 비밀번호) 파악이 어려움
- 사용자 간 WiFi 정보를 **크라우드소싱** 방식으로 공유하면 이 문제를 효과적으로 해결 가능

### 1.2 시장 규모 및 트렌드 (2025~2026)
- 전 세계 공공 WiFi 핫스팟 수는 지속적으로 증가 중
- **DePIN (Decentralized Physical Infrastructure Network)** 개념과 결합하여, 토큰 보상 기반의 인프라 공유 모델이 부상
- **eSIM** 서비스와의 통합이 새로운 수익 모델로 자리잡는 추세
- 개인정보 보호·보안에 대한 인식 증가로 **내장 VPN** 기능이 표준화

## 4. 비즈니스 모델 분석

### 4.1 주요 수익 모델

| 모델 | 설명 | 적용 사례 |
|------|------|-----------|
| **프리미엄 구독** | 광고 제거, 확장 DB 접근, 고급 오프라인 기능 | WiFi Map Pro |
| **광고** | AdMob 등 통한 인앱 광고 | WiFi Map (20~30% 수익) |
| **eSIM 판매** | 글로벌 모바일 데이터 패키지 판매 | WiFi Map, Instabridge |
| **토큰 이코노미** | 기여 보상 + 인앱 유틸리티 + 거버넌스 | WiFi Map ($WIFI) |
| **B2B 데이터** | 익명화된 이동 패턴·연결 데이터 제공 | 일부 DePIN 프로젝트 |
| **캐리어 오프로딩** | 통신사가 트래픽 분산 비용 지불 | DePIN 네트워크 |

### 4.2 DePIN 모델의 진화 (2025~2026)
- 초기 투기적 토큰 보상에서 → **실제 네트워크 사용량 기반 보상**으로 전환
- 토큰 보상이 실제 수요를 초과하면 인플레이션 → 실패로 인식
- **패시브 참여** 모델 선호 (백그라운드 대역폭 공유, 경량 하드웨어)
- 프라이버시 보존 기술 (ZKP) 표준화 추세
- 2026년은 크립토 네이티브 인프라 ↔ 전통 통신 모델의 **통합 원년**으로 평가

## 6. 확장 서비스 제안 🚀

기존 WiFi Finder의 한계를 넘어, 차별화된 가치를 제공할 수 있는 **6가지 확장 서비스 영역**을 제안합니다:

### 6.1 🏢 "일하기 좋은 장소" 통합 플랫폼

> WiFi 정보 + 작업 환경 정보를 결합

| 추가 정보 | 설명 |
|-----------|------|
| **콘센트 위치** | 좌석별 콘센트 유무 및 수량 |
| **소음 수준** | 시간대별 소음 레벨 (조용함 / 보통 / 시끄러움) |
| **좌석 유형** | 1인석, 그룹석, 스탠딩 데스크 등 |
| **영업시간** | 늦은 시간까지 이용 가능한 장소 필터 |
| **음료 가격대** | 아메리카노 기준 가격 (비용 대비 체류 시간) |

**타겟 사용자**: 디지털 노마드, 프리랜서, 원격근무자, 대학생

---

### 6.2 📊 실시간 네트워크 품질 모니터링

> 단순 "있다/없다"를 넘어, **품질 정보** 제공

- **자동 속도 측정**: 앱 사용 중 백그라운드에서 속도 테스트 → DB 자동 업데이트
- **시간대별 속도 그래프**: 피크 시간대 vs 한산한 시간대 속도 비교
- **안정성 점수**: 끊김 빈도, 연결 유지 시간 등 종합 평가
- **네트워크 혼잡도**: 접속 인원 추정 (RSSI 기반)

---

### 6.3 🎯 게이미피케이션 & 커뮤니티 시스템

> 사용자 참여를 지속시키는 동기부여 시스템

```
┌─────────────────────────────────────────────────┐
│  🏆 기여자 등급 시스템                            │
│  ─────────────────────────────────────────────── │
│  🔵 탐색자 (Explorer)    → 10개 핫스팟 등록       │
│  🟢 안내자 (Guide)       → 50개 핫스팟 + 리뷰 20 │
│  🟡 개척자 (Pioneer)     → 200개 + 지역 1위      │
│  🔴 마스터 (WiFi Master) → 500개 + 검증 100     │
│  💎 레전드 (Legend)      → 1000개 + 커뮤니티 공헌 │
└─────────────────────────────────────────────────┘
```

- **지역별 리더보드**: 내 동네 WiFi 기여 순위
- **탐험 퀘스트**: "이번 주 OO구 미등록 지역 3곳 등록하기"
- **뱃지 시스템**: 특수 업적 달성 시 뱃지 수여
- **포인트 → 리워드**: 제휴 카페 할인쿠폰, 데이터 충전 등

---

### 6.4 🔋 충전 인프라 통합

> WiFi + 충전 = 모바일 사용자의 핵심 니즈 동시 해결

- **보조배터리 대여소** 위치 정보
- **무선충전 가능 장소** 표시
- **콘센트 사용 가능 좌석** 실시간 정보
- **EV 충전소** 위치 (차량 사용자 확장)

---

### 6.5 🛡️ 보안 등급 시스템

> 공공 WiFi의 최대 우려사항인 **보안**을 해결

| 보안 등급 | 설명 | 표시 |
|-----------|------|------|
| **A (안전)** | WPA3 암호화, 기관 운영, 인증서 유효 | 🟢 |
| **B (양호)** | WPA2, 사업장 운영, 기본 보안 | 🟡 |
| **C (주의)** | 오픈 네트워크, 비밀번호 없음 | 🟠 |
| **D (위험)** | 의심스러운 AP, 사용자 신고 다수 | 🔴 |

- **피싱 AP 탐지**: 유사 SSID 경고 (예: "Starbucks_Free" vs "Starbuck_Free")
- **접속 전 보안 스캔**: 네트워크 기본 보안 수준 자동 체크
- **VPN 자동 활성화**: 보안 등급 C 이하 시 VPN 연결 권장/자동 실행

---

### 6.6 🤝 지역 상권 연계 서비스

> WiFi 제공 장소(카페, 식당 등)와의 **상생 모델**

- **사업장 프로필**: WiFi 제공 카페·식당의 메뉴, 분위기, 혼잡도 정보
- **쿠폰·프로모션**: WiFi 사용자 대상 할인 제공 (사업장 → 고객 유치)
- **예약 연동**: WiFi 있는 카페 좌석 예약 기능
- **리뷰 시스템**: WiFi 품질 + 장소 품질 통합 리뷰
- **사업장 대시보드**: 방문자 통계, 마케팅 인사이트 제공 (B2B 수익)

## 7. 경쟁 우위 전략 제안

### 7.1 한국 시장 특화 차별화 포인트

| 전략 | 설명 |
|------|------|

| **공공데이터 통합** | 공공데이터포털 WiFi API + 커뮤니티 데이터 하이브리드 |
| **네이버/카카오 연동** | 네이버 지도, 카카오맵 SDK 활용 (한국 사용자 친숙) |
| **지역 상권 연계** | 한국 자영업 생태계와의 시너지 (배달의민족처럼) |
| **통신사 제휴** | SKT/KT/LGU+ 데이터 절약 서비스로 포지셔닝 |

### 7.2 MVP (최소 기능 제품) 우선순위

```
Phase 1 (핵심)
├── 📍 지도 기반 WiFi 핫스팟 표시
├── 📝 사용자 핫스팟 등록/공유
├── 🔑 비밀번호 공유
├── ⭐ 별점/리뷰 시스템
└── 🔍 주변 검색 + 필터

Phase 2 (성장)
```
